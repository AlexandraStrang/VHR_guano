# Intra-seasonal variation and factors influencing satellite-derived population indices of Adélie penguin

## Overview:
This work represents the first chapter of my PhD research. We aimed to investigate the intra-seasonal variation and factors influencing VHR-derived population indices of Adélie penguins.

## Contains:
This repository contains the R code and data required for completing the analysis in Strang et al. (in prep).
